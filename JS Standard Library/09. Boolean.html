Boolean

Boolean merupakan wrapper class untuk tipe primitif boolean (beda Boolean dan boolean di huruf B/b)

Boolean memiliki method toString() untuk mengkonversi ke string

Dan memiliki method valueOf() untuk mengkonversi ke tipe boolean primitif

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean

Boolean.prototype.toString() Mengembalikan string yang berisi salah satu trueatau yang lain, falsetergantung pada nilai objek. Mengganti Object.prototype.toString()metode. Boolean.prototype.valueOf() Mengembalikan nilai primitif objek Boolean. Mengganti Object.prototype.valueOf()metode.
