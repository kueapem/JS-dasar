Reflect

Reflect merupakan class di js yang digunakan untuk mengeksekusi ja fungsi

Reflect tidak memiliki constuctor, dan cra penggunaan reflect tidak dengan membuat object dengan new reflect

Penggunaan reflect adalah menggunakan banyak sekali static methodnya

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Reflect

Properti statis Reflect[Symbol.toStringTag] Nilai awal properti [Symbol.toStringTag]adalah string "Reflect". Properti ini digunakan dalam Object.prototype.toString(). Metode statis Reflect.apply() Memanggil targetfungsi dengan argumen yang ditentukan oleh argumentsListparameter. Lihat juga Function.prototype.apply(). Reflect.construct() Operator sebagai fungsi. Setara dengan pemanggilan . Juga menyediakan opsi untuk menentukan prototipe yang berbeda new.new target(...argumentsList) Reflect.defineProperty() Mirip dengan Object.defineProperty(). Mengembalikan boolean jika trueproperti berhasil didefinisikan. Reflect.deleteProperty() Operator sebagai suatu fungsi delete. Setara dengan pemanggilan delete target[propertyKey]. Reflect.get() Mengembalikan nilai properti. Berfungsi seperti mendapatkan properti dari sebuah objek ( target[propertyKey]) sebagai sebuah fungsi. Reflect.getOwnPropertyDescriptor() Mirip dengan Object.getOwnPropertyDescriptor(). Mengembalikan deskriptor properti dari properti yang diberikan jika ada pada objek, undefinedjika tidak. Reflect.getPrototypeOf() Sama seperti Object.getPrototypeOf(). Reflect.has() Mengembalikan nilai boolean yang menunjukkan apakah target memiliki properti. Baik sebagai milik sendiri maupun warisan. Berfungsi seperti inoperator sebagai fungsi. Reflect.isExtensible() Sama seperti Object.isExtensible(). Mengembalikan nilai boolean truejika target dapat diperluas. Reflect.ownKeys() Mengembalikan array kunci properti milik objek target (bukan yang diwarisi). Reflect.preventExtensions() Mirip dengan Object.preventExtensions(). Mengembalikan boolean truejika pembaruan berhasil. Reflect.set() Fungsi yang menetapkan nilai ke properti. Mengembalikan nilai boolean truejika pembaruan berhasil. Reflect.setPrototypeOf() Fungsi yang menetapkan prototipe suatu objek. Mengembalikan nilai boolean truejika pembaruan berhasil.
