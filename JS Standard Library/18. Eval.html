Eval

Merupakan fungsi yang digunakan untuk eksekusi kode js dari string

Fitur ini menarik, namun perlu berhati-hati ketika menggunakannya

Jika sampai salah penggunaannya, maka bisa jadi kita malah mengeksekusi kode program yang bisa menyebabkan masalah keamanan di web kita
