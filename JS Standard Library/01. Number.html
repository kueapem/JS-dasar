Number

Merupakan fungsi yang digunakan untuk konversi ke tipe data number

JIka data tidak bisa di konversi ke number, secara otomatis hasilnya adalah NaN

Number

Kode:


const input = "123454";
const number = Number(input);

console.log(typeof input);
console.log(typeof number);

Number memiliki banyak static properties, seperti:

Number.MIN_VALUE untuk mendapatkan number minimal

Number.MAX_VALUE untuk mendapatkan number maksimal

static_properties

Number static method

Number memiliki banyak static method, seperti:

Number.isNaN(value) untuk cek apakah value adalah NaN atau bukan

Number.isInteger(value) cek apakah value berupa integer atau bukan

static_methods

Number.isFinite() Tentukan apakah nilai yang dilewatkan merupakan bilangan terbatas. Number.isInteger() Tentukan apakah nilai yang dilewatkan merupakan bilangan bulat. Number.isNaN() Tentukan apakah nilai yang dilewatkan adalah NaN. Number.isSafeInteger() Tentukan apakah nilai yang dilewatkan merupakan bilangan bulat aman (angka antara -(2,53 - 1) dan 2,53 - 1). Number.parseFloat() Ini sama dengan parseFloat()fungsi global. Number.parseInt() Ini sama dengan parseInt()fungsi global.
