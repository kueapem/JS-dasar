Strict mode

Dari chatgpt

Strict mode merupakan ditur js dimana kode berjalan lebih ketak. Ini membantu untuk menghindari kesalahan umum, menangkanp bug lebih awal, meningkatkan perporma di beberapa engine js

Cara aktipkannya cukup pakai


"use strict";

di paling atas dari file atau fungsi. penulisan harus berupa string pakai tanda kutip dan diletakkan di paling atas

Berikut beberapa perbedaan saat pakai strict mode:

1. Gak bisa pakai varibale tanpa let/const/atau var

2. Gak bisa duplikat parameter

3. Gak bisa ubah property read-only atua object non-writable

4. This di fungsi biasa jadi undefined (bukan window)

Kapan sebaiknya pakai strict mode

Saat belajar js, nulis function atau module, saat bikin project besar
