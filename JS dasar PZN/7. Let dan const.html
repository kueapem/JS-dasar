Variable Let dan const

Sejak tahun 2015, kata kunci var dianggap tidak lagi direkomendasikan digunakan lagi.

Sebagai gantinya kita bisa menggunkana let atau const.

Let

Let digunakan untuk menggantikan var yang memiliki beberap error.

Jika ingin deklarasikan lagi variablenya

Const

const artinya konstanta / tetap

kata kunci ini lebih spesifikasi, dan tidak bisa di deklarasikan ulang.
