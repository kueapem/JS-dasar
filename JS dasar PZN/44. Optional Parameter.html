Optional Parameter

Berbeda di bahasa programana lainnya seperti JAVA, di js paramter itu opsional

Artinya kita tidak wajib mengisi nilai nya ketika memanggil fungsi

Jika tidak ada nilainya yang kita kirimkan ke Parameter ketika memanggil fungsi, maka secara otomatis paramter akan bernilai undefined

Jika di bahasa programan lain ini akan error, namun karena js adalah parametrnya optional jadi secara default jika kita tidak menetapkan nilai paramternya akan output undefined
