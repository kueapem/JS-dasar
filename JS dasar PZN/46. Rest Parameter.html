Rest Parameter

fitur yang bisa digunakan untuk mengirimkan banyak data ke dalam 1 parameter dan akan otomatis di konversi jadi array

Untuk membuat rest paarmter ada ketentuannya

- Hanya boleh ada 1 dalam 1 function

- Hanya boleh diletakkan di paling akhir gak boleh di awal atau tengah, kecuali hanya ada rest paramter saja

- Di bahasa programan lain ada juga disebut variable argument
