Kata kunci this

Kata kunci this adalah referensi ke object milik siapa

Tiap lokasi kata this, bisa berbeda beda referensi pemiliknya

Dalam object method, this merupakan referensi ke object pemiliki functionya

Di global scope, this referensinya ke global object (di browser biasanya window)

Di function, this merupakan referensinya ke ghlobal object (dibrowser biasanay window)

Di fungsi dengan strict mode (akan dibahas nanti), this adalah undefined

Dalam event, this merupakan referensi ke element yang menerima event (dibahas di materi document oibject method)
