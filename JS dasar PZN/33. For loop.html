For Loop

For adlaah salah satu kata kunci yang bisa digunakan untuk melakukan loop / perulangan

Block kode yang ada didalam for akan di ulang selama kondisi for terpenuhi

Sintak for loop

for (init statement; kondisi; post statement){

// block perulangan

}

init statement -> akan di eksekusi sekali di awal loop

kondisi -> akan dilakukan pengecekan setiap kali loop dilakukan, jika true, loop akan di lakukan, jika false loop akan berhenti

post statement -> akan di eksekusi seiap kali di akhir loop

init, kondisi, post tidak wajib diisi, jika kondisi tidak diisi, berarti loop selalu bernilai true -> terjadilah infinity loop
