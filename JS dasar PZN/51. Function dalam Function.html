Fungsi didalam fungsi / nested functiopn

Kita bisa membuat fungsi didalam fungsi

fungsi yang ada didalam disebut innet function

inner function hanya bisa diakses ditempat kita membuat fungsi nya, tidak bisa diakses diluar fungsinya

Analoginya

Bayangkan ada dapur didalam rumah

rumah = fungsi utama

dapur = inner fungsi

dapur hanya bisa digunakan didalam rumah itu. Kita tidak bisa masuk dapur tanpa masuk ke rumahnya dulu.
