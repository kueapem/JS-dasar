Tipe Data array

Array adalah tipe data yang berisikan kumpulan data

Arrauy bersifat dinamis dalam js, artinya bisa bertambha sendiri datanya saat kita memasukkan data kedalam array.

Contoh array

let nama = []; // ini array kosong

let nama = ["Kenny", "Rui", "Tachibana", 23, 2023, 2232, true] // array dengan isinya

Sangat jarang menuliskan array dengan datanya langsung, Biasanya kode array ini digunakan untuk menambah, mengubah dan menghapus data array.

Sebelum itu, kikta pahami cara kerja array.

Array akan menyimpan data dalam berurutan, dimana urutan pertama dimulai dari 0

Setiap memasukkan data baru, akan mengisi urutan terakhir

Urutan di array disebut index

Operasi di array

Berikut ini daftar operasi yang bisa dilakukan dengan array

array.push(value) -> Menambahkan data baru ke array

array.length -> mendapatkan panjnag array

array[index] -> mengambil data index array

array[index] = value -> mengubah data index array

delete array[index] -> hapus data index array
