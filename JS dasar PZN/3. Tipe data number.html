Tipe data number

Tipe data adalah jenis data. Ex, aplikasi penjualan, ada nama barang, harga, dan sebagainya. Nah harga itu bisa disebut tipe data number.

di js hanya ada 1 tipe data number yang bisa berupa bilangan bulat (1,2,3,4,5 dst) atau desimal (0.1,0.2,0.22,9.2 dst)

Tipe di bahasa program lain biasanya membedakan tipe data number bilangan bulat dan desimal, namun di js itu disatukan.

artinya berbeda misal di python bilangan nya 7 ini disebut bulat atau integer, dan 9.9 ini float-poin (desimal).

Namun di js kedunya disebut number.

Js juga mendukung binary, hexadesiman, dan octal

Contoh binary didepannya ditambahkan depannya 0b

Contoh hexadesimal depannya 0x

Contoh octal depannya 0o
