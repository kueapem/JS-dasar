Konversi string dan number

Aplikasi kadang enerima input dari user berupo string bukan number ("4").

Sedangkan kita ingin data yang masuk itu berupa number

Maka disarankan untuk melakukan konversi tipe data

contoh kasus jika kita tidak melakukan konversi tipe data

Cara yang dilakukan untuk konversi

Ada beberapa cara untuk atasi masalah diatas dengan konversi string dan number

parseInt(string) -> konversi dari string ke number (bilangan bulat)

parseFloat(string) -> konversi dari string ke number (bilangan pecahan)

Number(string) -> konversi dari string ke number (bilangan bulat dan pecahan)

number.toString() -> konversi number ke string

a. ParseInt()

b. parseFloat()

c. Number()

d. toString

Kenapa gak pakai Number saja?

Pada bagian parseint dan float mungkin bisa diselsaiakan dengan hanay dengan number() saja ya, nmaun sejatinya gak gitu.

Mungkin yang sederhana bisa, namun jika sudah komplek gak bisa.

Misal ada user gak sengaja masukkan data input number dan teks, maka di number() itu akan dikembalikan jadi NaN, sedangkan di int dan float akan di ambil numbernya saja jika number yang diinput ada didepan teks (123teks, 1.222teks)
