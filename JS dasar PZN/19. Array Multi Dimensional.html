Array 1 dimensi

Array 2 dimensi

misal

let data = [["array1"], ["array2"], ["array3"]];

Artinya nilai "array3.1" akan masuk ke dalam array yang sebelumnya hanya berisi "array3".

Hasil akhirnya:
