Tipe data Object

Object mirip dengan array, tetapi key (index)-nya bisa berupa string yang kita tentukan sendiri, sedangkan array menggunakan index angka secara otomatis..

berbeda dengan array, indexnya sudah defautl, namundi object kita bebas memasukkan datany indndenya.

Di obejct index itu disebut sebagai property.

delete obj.property / delete obj[property] -> hapus property
