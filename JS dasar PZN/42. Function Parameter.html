Function parameter

Kita bisa kirimkan data ke function yang kita panggil

Untuk melakukannya, kita perlu menambahkan parameter atau argument di function yang sudah di buat

Parameter ditempatkan di dalam kurung () dideklarasi function

Parameter bisa lebih dari satu,jik lebih dari satu dipisahkan dengan koma

Analoginya gini

function buatNasiGoreng(nasi, telur, pedas) {

document.writeln("Membuat nasi goreng dengan " + nasi + ", " + telur + ", dan tingkat kepedasan " + pedas);

}

buatNasiGoreng("Nasi putih", "Telur ayam", "pedas sedang");

outputnya

Membuat nasi goreng dengan Nasi putih, Telur ayam, dan tingkat kepedasan pedas sedang
