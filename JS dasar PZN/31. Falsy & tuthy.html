Falsy dan truthy

a. falsy

adalah nilai ketika konteksnya dalam boolean berupa false

Ini fitur unik js yang berguna namun membingiungkan.

Jadi di js itu, kondisi itu tidak hanya terjadi di boolean, namun diluar boolean pun bisa, namun kita perlu tahu data / nilai yang dianggap falsy / false

Data dianggap falsy dintaranya:

false -> boolean false

0, -0 -> nol dan negative nol

"", '' -> string kosong

undefined

null

NaN

jadi intinya nanti kita bisa bermain boolean tanpa terikat dengan data / nilai true dan false doang, namun dengan data diatas juga sebagai aturannya

Data diatas dianggap false semua

b.truthy

Data yang bernilai true

Data yang dianggap true adalah data selain yang bernilai falsy / false (kebalian falsy)
