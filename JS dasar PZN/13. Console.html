Console

Di js kita bisa membuat logging bernama console

logging adalah sebuah mekanisme yang baisa dilakukan programmer untuk menampilkan inf dari aplikasi yang berjalan tanpa mengganggu proses aplikasi yang berjalan tersebut.

Di js logging mnggunakan console, yang biasanya akan tampil di browser console.

di js sendiri ada beberapa console methode yang bisa digunakan.

console.info(...) -> menampilkan sebagai informasi

console.log(...) -> menampilkan logging, sama aja dengan consolee info

console.warn(...) -> menampilkan warning

console.error(...) -> menampilkan error

console.table(...) -> menampilkan dalam bentuk tabel

untuk aktipkan mode debug, klik deafult level -> aktipkan verbose
