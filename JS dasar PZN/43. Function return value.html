Function return value

Secara dedault fungsi itu tidak menghasilkan value apapun, namun jika kita ingin, kita bisa membuat fungsi mengembalikan value

Agar function menghasilkan value, kita bisa menggunakan kata kunci return didala functionya

Dan dialam block function, untuk menghasilkan nilai tersebut, kita harus menggunakan kata kunci return, lalu di ikuti dengan data yang ingin dihasilkan

Function hanya bisa mengembalikan satu data, jika kita ingin mengembalikan beberapa data sekaligus, kita bisa menggunakan array sebagai return valuenya.

Contoh Analoginya

function pesanMakanan(menu) {

return "Ini pesanan Anda: " + menu;

}

let makanan = pesanMakanan("Nasi Goreng");

console.log(makanan); // Output: Ini pesanan Anda: Nasi Goreng

Analogi lainnya

function tarikUang(saldo, jumlahTarik) {

if (jumlahTarik > saldo) {

return "Saldo tidak cukup!";

}

saldo - jumlahTarik; // ATM mengembalikan saldo baru

}

let sisaSaldo = tarikUang(500000, 200000);

console.log(sisaSaldo); // Output: 300000
