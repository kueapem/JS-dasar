Operator logika

operator logika adalah operasi dari 2 nilai boolean

hasilnya adalah boolean lagi

Op logika di Js ada 3

&& -> dan

|| -> atau

! -> kebalikan

&&

true && true = true

true && false = false

false && true = false

false && false = false

Contoh

||

true || true = true

true || false = true

false || true = true

false || false = false

contoh

!

Op ini juga bisa disebut unary karena hanya butuh 1 data

true ! false

false ! true

contoh
