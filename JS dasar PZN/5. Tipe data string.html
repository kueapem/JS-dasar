Tipe data string

Tipe data ini bisa disebut tipe data teks yang berisi data kosong atau karakter.

Untuk membuat data string, kita perlu menggunakan petik 2 atau 1. (""/'') sebelum dan setelah teksnya.

Ada juga kasus gini, number yang ternyata adalah string.

Jika kita masukkan tipe data number ke dalam tanda petik, maka jadinya adalah string.

Menambah string

Kita bisa gabungkan beberap string / lebih dari 1 string menggunkana operator + (plus)

misal "nama" + "kenny"

cara ini akan sangat menarik saat kalian belajar variable nanti.

Escape sequence

js mendukung escape sequence dio string. Escape sequensce adalah karakter khusus seperti TAB, enter, tanda petiik, dll.

\n Enter
\t Tab
\' Petik 1
\" Petik 2
\\ \
