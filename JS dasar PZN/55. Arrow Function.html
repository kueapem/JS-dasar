Arrow Function

Merupakan alternative fungsi yang lebih sederhana dari fungsi biasanya

Namun terdapat limitasi dan juga tidak bisa digunkana di semua situasi

dinamakan arrow fungsi karena menggunakan => (seeprti panah)

Berikut beberapa kekurangan arrow fungsi

  • tidak memiliki fitur argument obejct (memnag sudah tidak direkomendasikan lagi, sekaarng pakai rest paarmeter)
  • tidak bisa menggunakan fungsi generator
  • tidak bisa mengakses this (yang nanti dibahas di fungsi object)
  • tidak bisa mengakses super (yang nanti dibahas di javascript OOP)

Arrow function tanpa block

Jika arrow fungsinya sederhana hanya 1 baris

kita bisa membuat arrow fungsi tanpa harus menggunakan block {}

Arrow fungsi return value

Arrow function juga bisa mengembalikan value, sama seperti fungsi biasnya

Jika menggunakan block, maka kita perlu kata kunci return

jika tidak ada block kita tidak perlu menggunakan kata kunci return

Arrow fungsi tanpa kurung parameter
