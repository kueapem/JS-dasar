Arrow fungsi di object

Arrow fungsi juga bisa kita gunakan di obeject method

Nmaun perlu dikethau, arrow fungsi tidak bisa digunkana untuk mengakses argument obejct, fungsi generator, kata kunci this, dan kata kunci super (yang dibahas di js oop)

Jadi pastikan gunakan arrow fungsi hanya memang ketika kita tidak butuh fitur fitur diatas

Arrow function bisa digunakan sebagai method di object, tetapi tidak disarankan jika kita membutuhkan akses ke fitur seperti this, arguments, super, atau ingin membuat function generator. Karena arrow function tidak memiliki this sendiri, maka method yang butuh akses ke object-nya sendiri sebaiknya tetap menggunakan function biasa.

🧠 Analogi: Arrow Function vs Function Biasa

Bayangkan kamu sedang di sebuah rumah yang punya beberapa anggota keluarga.

Setiap anggota bisa memanggil nama mereka sendiri lewat this.

🔧 Function Biasa (Orang yang kenal dirinya)


const orang = {
	nama: "Dina",
	sapa: function () {
		console.log(`Halo, saya ${this.nama}`);
	}
	};
	
orang.sapa(); // Halo, saya Dina

📝 Analogi:

Function biasa itu kayak Dina sendiri yang bilang, “Halo, saya Dina.”

Dia tahu siapa dirinya (this.nama), karena dia bagian dari rumah itu (object orang).

🏹 Arrow Function (Orang yang hanya meniru dari luar)


const orang = {
	nama: "Dina",
	sapa: () => {
		console.log(`Halo, saya ${this.nama}`);
	}
	};
	
orang.sapa(); // Halo, saya undefined

📝 Analogi:

Arrow function itu kayak tetangga Dina yang datang ke rumah dan disuruh bilang, “Halo, saya ...”

Tapi si tetangga nggak tahu dia lagi di rumah siapa. Jadi dia cuma meniru dari tempat asal dia (lingkungan luar), dan akhirnya malah bilang “Halo, saya undefined”, karena dia nggak punya this.nama.

📌 Kesimpulan Analogi:

Function biasa = orang asli di dalam rumah → tahu siapa dirinya → bisa akses this.

Arrow function = orang luar (cuma numpang ngomong) → nggak tahu rumah siapa → this jadi kacau.
