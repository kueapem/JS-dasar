Variable

variable adalah tempat menyimpan data

dengan kita menyimpan data di variable, kita bisa menggunakan / menyebutkan nama varibale berulang-ulang, cukup tuliskan nama variablenya

Untuk membuat variabel, gunakan kata kunci var diikuti nama variabelnya

js adalah dynamic language, artinya variabel tidak terpaku pada satu tipe data, kita bisa ubah tipe data di variable yang sama, artinya kita bisa masukkan nila tipe data number, string, boolean dan sebagainya bebas saja.

kita juga bisa mengganti nilai dari variabelnya
