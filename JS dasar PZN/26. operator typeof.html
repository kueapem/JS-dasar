Operator typeof

merupakan operator yang bisa kita gunakan untuk melihat tipe data sebuah value atau variable

karena js dynamic language, jadi kita perlu cek tipew datanya

hasil dari op ini adlaha string tipe datanya

Tipe Data Contoh Hasil dari typeof
Number 42, 3.14 "number"
String "Hello" "string"
Boolean true, false "boolean"
Undefined let x; "undefined"
Object {name: "John"}, null "object"
Function function() {} "function"
Symbol Symbol("id") "symbol"
BigInt 123n "bigint"
