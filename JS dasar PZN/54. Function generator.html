Function generator

merupakan fungsi yang digukana untuk mmebuat data generator

generator merupakan sebuah data yang bisa di iterasi seperti array

untuk membuat generator kita perlu tanda * setelah kata function

dan untuk mengembalikan data ditiap iterasi, kita bisa gunakan kata kunci yield diikuti datanya

Ingat

Data generator hanay bisa diiterasi, tidak bisa di akses ebrupa index bahkan diubah datanya gak bisa
