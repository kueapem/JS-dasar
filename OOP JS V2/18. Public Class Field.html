üìò Apa itu Public Class Field?

Ini adalah fitur baru di JS modern ES2022 yang mempermudah kit dekalarasikan properti langsung didalam class tanpa harus melalui constructor

Sebelumnnya, kita biasa menuliskan properti begini didalam constructor


class Orang {
	constructor() {
		this.nama = "Budi";
	}
}

Dengan public class field, kamu bisa menulisnya langsung di luar constructor:


class Orang {
	nama = "Budi";
}

Efeknya sama seperti Sebelumnnya: setiap instance Orang akan memiliki properti dengan nilai "Budi".


class Mobil {
	warna = "merah"; // public class field

	suara() {
		console.log("Brummm!");
	}
}

const m = new Mobil();
console.log(m.warna); // merah
m.suara();            // Brummm!

üéØ Analoginya untuk pemula:

Bayangkan kamu membuat cetakan kue donat (class). Kalau kamu selalu ingin setiap donat punya taburan gula saat di cetak, kamu bisa tentukan langsung.


class Donat {
	taburan = "gula";
}

Kalau pakai constructor, kamu seperti memberikan gula satu persatu ketiap donat. Tapi dengan public class field, kamu udah langsung sematkan gula dari cetakannya.

Versi PZN

Biasanya, saat kita ingin menambahkan field (properti yang berisi value), kita biasanya tambahkan di constructor

Namun ada proposal di ES yang mengajukan pembuatan public class field ditempatkan di luar constructor, selevel dengan penempatan method

Proposal ini masil belum final, namun beberapa browser sudah mendukungnya

Public class field

Dalam proposal tersebut juga disebutkan bahwa ecmascript akan mendukung access modifier public dan private

public artinya bisa di akses di luar class, dan private hanya bisa diakses dari dalam class saja

Private class field akan dibahasa di chap selanjutnya

Untuk membuat public class field, kita bisa langsung buat nama field dengan value nya selevel dengan method

Jika kita tidak memasukkan value ke filed tersebut, maka field tersebut memiliki nilai undefined.


class Person {
	firstName;
	lastName;
	balance = 0;
}

const i = new Person();
console.log(i);

Kita bisa mengubah nilai public class field dari dalam constructor


class Person {
	firstName;
	lastName;
	balance = 0;

	constructor(firstName, lastName) {
		this.firstName = firstName; // this.firstName dari public class
		this.lastName = lastName; 
	}
}

const i = new Person("Kenny", "ken");
console.log(i);
