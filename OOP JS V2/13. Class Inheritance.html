Class Inheritance

Versi chatgpt

üìò Apa Itu Inheritance?

Inheritance (pewarisan) artinya kita bisa membuat class baru (anak) yang mewarisi semua properti dan method dari class lain (induk)

.

Class induk = parent class (atau superclass)

Class anak = child class (atau subclass)

Dengan Inheritance

Kita bisa menggunakan ulang kode dari class induk

Kita bisa menambhakan atau memodifikasi method di class anak

üß† Analogi Sederhana

Bayangkan kita punya class Kendaraan:

Semua kendaraan punya roda dan bisa jalan

Lalu kita buat class Mobil:

Mobil adalah kendaraan, jad dia juga punya roda dan bisa jalan, tapi dia juga punya fitur khusus seperti AC, klakson, dll

Contoh


// Class induk
class Kendaraan {
	constructor (jenis) {
		this.jenis = jenis;
	}

	jalan() {
		console.log(`${this.jenis} sedang berjalan`);
	}
}

// Class anak mewarisi dari class Kendaraan
class Mobil extends Kendaraan {
	constructor(jenis, model) {
		super(jenis); // panggil constructor parent-nya
		this.model = model;
	}

	klakson() {
		console.log(`${this.model} membunyikan klakson`);
	}
}

// Membuat object dari class anak
let avanza = new Mobil("Mobil", "Avanza");

avanza.jalan();    // Output: Mobil sedang berjalan
avanza.klakson();  // Output: Avanza membunyikan klakson!

üîç Penjelasan:

extends digunakan agar Mobil mewarisi dari kendaraan

super(class) didalam constructor Mobil digunakan untuk memanggil constructor class induk, agar properti jenis bisa di set

Mobil bisa menggunkaan method jalan() dari kendaraan karena dia mewarisinya

üß© Kenapa Inheritance Berguna?

Mengurangi pengulangan kode (DRY = Don‚Äôt Repeat Yourself).

Membuat struktur kode lebih rapi dan mudah dikembangkan.

Mempermudah ekspansi fitur, cukup dengan override atau extend method dari parent.

Tambahan

Urutan Penulisan parameter constructor(jenis, model)

penulsiannya tidak wajib berurutan, bisa gini juga (model, jenis)

Penulisan super(jenis); harus yang pertama

Pada kata kunci super(jenis); harus dituliskan paling awal, tidak boleh kedua tau terakhir. 


// contoh benar
super(jenis); 
this.model = model;

// contoh salah
this.model = model;
super(jenis);

Sebelumnya kita tahu bahwa di prototype mendukung Inheritance pewarisan, meskipun agak tricky cara membuatnya

Untungnya di ES6 sudah di perbaiki dengan fitur class

Sebuah class bisa melakukan pewarisan dari class lain dengan kata kunci extends

Di js, class Inheritance sama seperti prototype Inheritance, hanya bisa memiliki satu parent class

Kode: class Inheritance(1)


class Employee {
	constructor(name) {
		this.name = name;
	}

	sayHello(name) {
		console.info(`Hi ${name}, my name is employee ${this.name}`);
	}
}

class Manager extends Employee{
	constructor(name) {
		this.name = name;
	}

	sayHello(name) {
		console.info(`Hi ${name}, my name is manager ${this.name}`);
	}
}
