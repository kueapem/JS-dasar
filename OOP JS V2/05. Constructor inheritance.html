ðŸ§  Konsep Inheritance (Pewarisan)

Inheritance artinya mewariskan property dan method dari constructor function lain.

Misalnya kamu punya Person, dan ingin buat Student yang punya data dari Person, ditambah data khusus seperti jurusan

âœ… Langkah-langkah Inheritance Versi Lama (tanpa class)

1. Buat constructor utama:


function Person(nama, umur) {
	this.nama = nama;
	this.umur = umur;
}

2. Buat constructor turunan:


function Student (nama, umur, jurusan) {
	Person.call(this, nama, umur); // warisan dari Person
	this.jurusan = jurusan;
}

3. Tes


const kenny = new Student("Kenny", 23, "Informatika");
console.log(kenny);

call() dipakai untuk menjalankan fungsi Person didalam konteks this miliki Student

Jadi, property nama dan umur akan "diisi" didalam this milik Student

call() Waktu kamu bikin constructor inheritance versi lama, kamu butuh call() supaya constructor induk (Person) bisa mengisi this milik anak (Student).

Versi PZN

Dalam constructor kita biasanya membuat property baik itu berisi nilai ataupun function

Didalam Constructor, kita bisa memanggil constructor lain, dengan begitu kita bisa mewarisi semua property yang di buat constructor lain tersebut

Untuk memanggil constructor lain, kita bisa menggunkan NamaConstructor.call(this, parameter)
