üìò Apa itu Static Method?

Static method adalah method yang:

Hanya bisa diakses lewat class, dan bukan lewat instance-nya (object)

Umumnya digunakan untuk fungsionalitas umum / global yang tidak perlu data dari instance

Sintak:


class Kalkulator {
	static tambah (a, b) {
		return a + b;
	}
}

console.log(Kalkulator.tambah(1, 1)); // 2
Kita tidak perlu new Kalkulator(). Karena tambah() adalah method statis

‚ùå Tidak Bisa Dipanggil dari Instance:


const calc = new Kalkulator();
calc.tambah(1, 1) // ERROOR: calc.tambah is not function

üß† Kapan Gunakan Static Method?

Gunakan static method untuk hal-hal berikut:

Utility function (konversi, format, hitung)

Fungsi-fungsi umum yang tidak tergantung pada data instance

Versi PZN

Kata kunci static juga bisa kita gunakan di method

Jika kita tambahkan di method, artinya method tersebut jadi miliki class nya, bukan prototype

Dan untuk akses method tersebut, kita juga bisa menggunakan cara static class field


class MathUtil {
	static sum(...numbers) {
		let total = 0;
		for (const number of numbers) {
			total += number;
		}
		return total;
	}
}

const sum = MathUtil.sum(1,1,1,1,1);
console.info(sum);
