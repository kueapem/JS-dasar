Kadang ada kasus kita ingin mengecek apakah sebuah object merupakan instance dari class tertentu atau bukan

Kita tidak bisa menggunakan operator typeof, karena object dari class, jika kita menggunkana typeof maka hasilnya adalah object

Operator instanceof akan menghasilkan boolean, true jika benar object tersebut adalah instance obejcet nya, flase jika salah.


class Employee {

}

class Manager {

}
let man1 = new Manager();
let em1 = new Employee();
console.log(man1 instanceof Manager);

Operator Instanceof di class inheritance

Operator instanceof mendukung class inheritance, artinya instanceof juga bisa digunakan untuk mengcek, apakah sebuah obeject adalah insatance dari class tertentu, atau turunan dari class tertentu


class Employee {

}

class Manager extends Employee {

}
let man1 = new Manager();
let em1 = new Employee();
console.log(man1 instanceof Manager);
