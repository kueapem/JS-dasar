Parameter di constructor function

Karena dalam Js, class adalah berbentuk function, jadi secara default, fungsi tersebut bisa memiliki paramter

constructor fungsi sama seperti function biasanya, bisa memiliki parameter, hal ini membuat ketika kita membuat object, kita bisa mengirim langsung sata lewat parameter di constructor function tersebut.
