üß© Apa itu prototype?

Di js, setiap fungsi (termasuk constructor function) secara otomatis punya properti prototype. Propertu ini digunakan untuk menyimpan method atau properti yang ingin diwariskan ke semua object yang dibuat dengan constructor itu.


function Person(nama, umur) {
	this.nama = nama;
	this.umur = umur;
}

// Menambahkan method ke prototype
Person.prototype.sayHello = function () {
	console.log(`Halo, aku ${this.nama}, umurku ${this.umur}`);
}

const kenny = new Person("Kenny", 23);
kenny.sayHello(); // Halo, aku kenny, umurku 23.

üß† Kenapa pakai prototype, bukan langsung di constructor?

Kalou bikin langsung method langsung di constructor kaayk gini:


function Person(nama, umur) {
	this.nama = nama;
	this.umur = umur;

	// ! method ini dibuat ulang di setiap instance
}

Setiap kali kamu membuat object baru, fungsi sayHello akan dibuat ulang, boros memori.

Sementara kalau paaki prototype, semua object berbagi method yang sama.


Person.prototype.sayHello = function () {
	console.log(`Helo, aku ${this.nama}`);
};

üîß Kesimpulan

Gunakan prototype untuk menambahkan method agar tidak boros memori.

Semua object yang dibuat dari constructor function akan me-warisi method yang ditambahkan ke .prototype.

Ini adalah dasar dari konsep pewarisan (inheritance) di JavaScript sebelum ada class.

Prototype inheritance

Saat kita membuat object dari constructor function, object tersebut di sebut instance, semua properti (baik itu value atau method), akan berada didalam instance objectnya

Setiap kita membuat sebuah constructor function, maka akan secara otomatis dibuatkan prototype nya, misal kita buat constructor Person, maka akan ada Person.prototype

Saat kita membuat sebuah object instance, secara otomatis object tersebut adalah inheritance / turunan dari constructor.protorype nya

Instance (dalam konteks OOP dan JavaScript) adalah objek nyata yang dibuat dari sebuah blueprint (cetakan), yaitu constructor function atau class.


function Robot(nama) {
	this.nama = nama;
}
const robot1 = new Robot("Robo-Kenny");

Nah, robot1 inilah yang disebut instance dari Robot.

Untuk askes prototype milik sebuah instance, kita bisa menggunakan __proto__
