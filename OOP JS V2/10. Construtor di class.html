Versi Chatgpt

Apa itu Constructor?

Constructor merupakan metode khusus di dalam class yang dipanggil otomatis saat kita membuat object baru.

Fungsinya adalah untuk menginisialiasasi atau memberikan nilai awal pada properti-properti object.

Analogi

Bayangkan kamu adalah pabrik mobil

Setiap mobil yang keluar dari pabrik harus punya warna dan model kan?

Nah didunia nyata:

Class itu seperti cetakan atau template mobil

Constructor itu seperti bagian pabrik yang mengisi mobil baru dengan warna dan model saat mobil dibuat

Kalau tidak ada Constructor, mobil-mobil itu keluar kosong, tidak punya warna, tidak punya model.

Constructor-lah yang mengisi mobil baru dengan atribut-atribut pentingnya.

Sintaks Constructor

Didalam class, kita mendefinisikan Constructor dengan menggunakan kata kunci Constructor diikuti tanda kurung () untuk parameter, dan didalamnya kita bisa menggunakna this untuk mengatur nilai properti object.


class Kendaraan {
	constructor(warna, model) {
		this.warna = warna;
		this.model = model;
	}
}

Penjelasan:

Constructor(warna, model): adalah constructor yang menerima dua parameter (warna dan model)

this.warna dan this.model: this meruuk pada object yang sedang dibuat. Dengan this.warna dan this.model, kita menetapkan nilai yang diberikan pada paramter warna dan model ke properti object tersebut.

Kenapa Menggunakan Constructor?

Constructor membantu kita untuk memastikan bahwa setiap object yang dibuat memiliki properti yang sudah terisi dengan nilai yang sesuai. Tanpa constructor, kita perlu mengatur nilai properti secara manual setelah object dibuat.

Contoh Penggunaan Constructor


class Kendaraan {
	constructor(warna, model) {
		this.warna = warna;
		this.model = model;
	}
}

let mobil1 = new Kendaraan("Merah", "Sport");
let mobil2 = new Kendaraan("Hijau", "SUV");

console.log(mobil1.warna); 
console.log(mobil2.model);

Kesimpulan

Constructor adalah metode yang dipanggil otomatis saat membuat object

Fungsi utamanta adalah untuk menginisialiasasi nilai object yang baru dibuat

Kita bisa memebrikan parameter pada constructor untuk memberikan nilai properti saat object dibuat.
