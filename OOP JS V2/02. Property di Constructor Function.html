Property di constructor fungsi

Sebenarnya setelah kita membuat object, kita bisa dengan mudah menambahkan property ke dalam object tersebut hanya dengan menggunakan nama variable nya, diikuti tanda titik dan nama property nya

Namun jika seperti itu, alhasil, constructor fungsi yang sudah kita buat tidak terlalu berguna, karean property nya hanya ada didalam object yang kita tambahkan property

Untuk menambahkan property didalam semua object yang dibuat dari constructor function, kita bisa menggunakan kata kunci this, lalu diikuti dengan nama property nya.
