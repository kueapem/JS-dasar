15. Super Method

Versi chatgpt

ðŸ”¹ Apa itu super (method)?

super digunakan untuk memanggil method dari class induk (parent) dari dalam class anak (child)

Kalau sebelumnya super(...) memanggil constructor, sekarang super.namaMethod(..) dipakai untuk memanggil method biasa di parent

ðŸ“¦ Analogi sederhana:

Bayangkan kamu (anak) mewarisi toko dari orang tuamu.

Kadang kamu ingin menambahkan fitur baru di kasir (method di anak), tapi tetap ingin memakai fungsi lama dari orang tuamu juga (method di parent).

Nah, kamu tinggal bilang:

"Pakai cara lama dulu deh (super.methodLama()), lalu aku tambahkan cara baru."

âœ… Kode Super Method


class Hewan {
	suara() {
		console.log("Hewan mengeluarkan suara umum.");
	}
}
	
class Kucing extends Hewan {
	suara() {
		super.suara(); // panggil method 'suara' dari parent
		console.log("Kucing mengeong: Meong!");
	}
}
	
const kucing1 = new Kucing();
kucing1.suara();

ðŸ§  Penjelasan langkah demi langkah:

Kucing mewarisi method suara() dari hewan

Tapi Kucing menimpa (override) method itu dengan versinya sendiri

Didalam method suara() milik Kucing, kita memanggil super.suara() agar suara dari Hewan tetap ikut dijalankan

Lalu, ditambahkan suara khas Kucing

ðŸŸ¢ Kapan super.method() digunakan?

Ketika kamu override method, tapi masih butuh sebagian logika dari parent.

Atau ketika kamu ingin menambah fungsi tambahan, bukan sepenuhnya mengganti.

Tambahan

Method di child class perlu memakai nama yang sama dengan method di parent class jika ingin mengganti (override) perilakunya. Kecuali kalau mau menambahkan method baru (gak perlu super.method() juga sih ya).



// dari parent
suara() {
	console.log("Hewan itu mengeluarkan suaranya");
}

// dari child
suara() {
	super.suara(); // panggil method 'suara' dari parent
	console.log("Kucing mengeong: Meong!");
}

Jika "Parent punya banyak method, tapi child cuma butuh satu": Kamu bisa pilih dan panggil hanya method yang dibutuhkan saja lewat super.namaMethod().

Jika "Child ingin pakai semua method dari parent": Tidak perlu override apapun kalau kamu tidak mau mengubah fungsinya. Semua method dari parent otomatis bisa dipakai langsung oleh child!


class Kucing extends Hewan {
	// Tidak override apapun
}

Versi PZN

Selain digunakan untuk memanggil constructor milik parent class, kata kunci super juga bisa digunkaan untuk mengakses method parent class

Caranya bisa menggunakan super.namaFungsinya()

Dengan kata lain, super sebenarnya adalah reference ke parent prototype, mirip seperti __proto__


class Shape {
	paint() {
		console.log("Paint shape");
	}
}

class Circle extends Shape {
	paint() {
		super.paint() // memanggil paint() method parent class
		console.log("Paint circle")
	}
}
