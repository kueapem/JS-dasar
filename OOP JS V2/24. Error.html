Throw Error

ðŸš¨ Apa itu throw?

throw adalah perintah untuk melempar error secara manual

Artinya: kamu bisa menghentikan eksekusi program sengaja, saat terjadi kondisi yang tidak sesuai

ðŸ§  Analogi:

Bayangkan kamu buat sistem kasir

Kalau jumlah bayar kurang dari total belanjaan, kamu mau munculkan peringatan dan hentikan proses

Disinilah kamu bisa pakai throw untuk melempar error, misalnya:

"Uang kurang, tolong bayar lebih!!!"


function bagi(a, b) {
	if (b === 0) {
		throw new Error("Tidak bisa dibagi dengan nol")
	}
	return a / b;
}

console.log(bagi(10, 0)); // Program berhenti di sini
console.log("Baris ini tidak akan dijalankan"); // Tidak pernah muncul

Throw Error

Saat kita membuat instance object dari class error, tidak lantas otomatis terjadi error

Kita perlu memberitahukan program kita, bahwa kita akan mentrigger sebuah error terjadi, atyau istilahnya adalah melempar error (throw)

Untuk melempar error, kita bisa menggunakan kata kunci Throw, diikuti dengan instance object error nya

Jika terjadi error, maka otomats kode program kita akan terhenti, dan kita bisa lihat detail errornya diconsole di aplikasi browser


class MathUtil {
	static sum(...numbers) {

		if (numbers.length === 0) { 
			throw new Error("total parameter harus lebih dari 0")
		}
		let total = 0;
		for (const number of numbers) {
			total += number;
		}
		return total;
	}
}

const sum = MathUtil.sum();
console.info(sum);
