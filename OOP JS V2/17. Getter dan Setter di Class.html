17. Getter dan Setter di Class

Versi chatgpt

📘 Apa itu Getter dan Setter?

Getter dan setter adalah cara membuat akses lebih aman dan terkontrol terhadap properti di dalam object (terutama di class)

Getter: untuk mengambil (read) data dari properti

Setter: untuk mengatur (write) data ke properti

🔧 Kenapa Butuh Getter/Setter?

Karena kamu kadang ingin:

🔐 Membatasi bagaimana data bisa diubah.

✅ Validasi data sebelum menyimpannya.

🙈 Menyembunyikan properti sebenarnya, tapi tetap memberi akses.

👶 Analogi Pemula:

Bayangkan kamu punya lemari (class), dan ada laci khusus (properti) bernama "nama":

Getter: kamu buka laci dan ambil isinya, tapi tetap rapi.

Setter: Kamu masukkan sesuatu kedalam laci, tapi kamu bisa tentukan "harus bersih dulu" atau "tidak boleh kosong"

✍️ Contoh Sederhana:


class Orang {
	constructor(nama) {
		this._nama = nama; // pakai _ untuk simpan data "asli"
	}
	
	// Getter
	get nama() {
		return this._nama.toUpperCase(); // tampilkan versi kapital
	}
	
	// Setter
	set nama(namaBaru) {
		if (namaBaru.length < 3) {
			console.log("Nama terlalu pendek!");
		} else {
			this._nama = namaBaru;
		}
	}
}
	
const orang = new Orang("andi");

console.log(orang.nama); // ANDI → lewat getter

orang.nama = "al";       // Nama terlalu pendek! → lewat setter
orang.nama = "budi";     // Setter valid, _nama diubah

console.log(orang.nama); // BUDI

💡 Catatan Penting:

Getter / setter tidak dipanggil pakai tanda kurung, tapi tetap terlihat seperti properti:


orang.nama   // bukan orang.nama()

Biasanya proeprti asli disimpan dengan awalan _ (seperti _nama) untuk dibedakan darei getter / setter -nya

Versi PZN

Class juga mendukung getter dan setter

Perlu diingat, getter dan setter ini akan berada di prototype, bukan di instacne object


class Person {
	constructor(firstName, lastName) {
		this.firstName = firstName;
		this.lastName = lastName;
	}

	get fullName () {
		return `${this.firstName} ${this.lastName}`;
	}

	set fullName(value) {
		const result = value.split(" ");
		this.firstName = result[0];
		this.lastName = result[1];
	}
}
