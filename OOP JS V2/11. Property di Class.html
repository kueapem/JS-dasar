Property di Class

Versi chatgpt

Apa itu Property di Class?

Property adlaah data atau sifat yang dimiliki object yang dibuat dari class

Kalau tadi kita sudah kenalan dengan constructor untuk mengisi property, sekarang kita fokus ke apa itu proprety itu sendiri

Cara Mendefinisikan Property

Biasanya,. kita Mendefinisikan property didalam constructor dengan menggunakan this, misalnya


class Mobil {
	constructor(warna, model) {
		this.warna = warna; // warna adalah property
		this.model = model; // model juga property
	}
}

this.warna dan this.model adalah property

this artinya "object ini sendiri" disini artinya adalah Mobil

Saat kita membuat objek baru, property ini akan otomatis tersedia di objek itu.

Contoh Penggunaan

Artinya:

Objek mobilBaru sekarang punya property:

warna: "Merah"

model: "Sedan"

Analogi Property

Kalau tadi di pabrik mobil:

Mobil itu (objek) punya sifat-sifat (property), misalnya warna dan model.

Jadi property itu kayak spesifikasi dari barang yang dibuat.

Contoh nyata lainnya:

Barang Properti
Mobil warna, model, kecepatan
Manusia nama, umur, tinggi badan
Laptop merk, RAM, kapasitas SSD

Ringkasan

Property = Data atau atribut yang dimiliki objek.

Biasanya dibuat di constructor pakai this.propertyName = value.

Property itu kayak identitas atau spesifikasi dari objek.
