Apa itu class di js

Class di JavaScript adalah template atau pola untuk membuat objek-objek. Dengan menggunakan class, kamu bisa mendefinisikan properti (sifat) dan method (perilaku) yang dimiliki oleh objek.

Menulis Class di JavaScript

Untuk membuat class di JavaScript, kita menggunakan kata kunci class, diikuti dengan nama class. Di dalam class, kita bisa mendefinisikan properti dan method yang dimiliki objek.

Sturktu dasar class


class Kendaraan {
	// Ini adalah class, tempat mendefinisikan properti dan method
}

Ini adalah dasar dari sebuah class. Kendaraan di sini adalah nama class yang bisa kita sesuaikan sesuai kebutuhan.

Kesimpulan:

Class adalah template untuk objek.

Kita bisa mendefinisikan properti (sifat) dan method (perilaku) di dalamnya.
