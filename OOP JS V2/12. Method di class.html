Method di class

versi chatgpt

Method adalah fungsi (function) yang menempel di dalam class, dan biasanya dipakai oleh object untuk melakukan sesuatu

Kalau property itu sebagai data (misalnya warna, model), maka method adalah aksi atau prilaku (misalnya jalan, berhenti)

Contoh


class Mobil {
	constructor (warna, model) {
		this.warna = warna;
		this.model = model;
	}

	nyalakan() {
		console.log("Mobil dinyalakan!");
	}

	info() {
		console.log(`Mobil ${this.model} berwarna ${this.warna}`);
	}
}

let mobilBaru = new Mobil("Merah", "Sedan");
mobilBaru.nyalakan(); // Output: Mobil dinyalakan!
mobilBaru.info();     // Output: Mobil Sedan berwarna Merah

Penjelasan:

nyalakan() dan info() adalah method

Kita memanggil lewat object mobilBaru.nyalakan()

Analogi method:

Bayangkan class Mobil itu seperti template mobil di pabrik:

property = bagian dari mobil itu (warna dan model)

Method = apa yang bisa dilakukan oleh mobil itu (nyalakan, membunyikan klakson dll)

Versi PZN

Membuat method di class sebenarnya bisa dilakukan dengan cara seperti menambahkan method di constructor fungsi

Namun, hal tersebut sebenarnya menambahkan method ke dalam instance object

Khusus untuk method sebaiknya kita menambahkannya ke dalam prototype, bukan ke instance object

Untuungnya di class, ada cara mudah menambahkan method secara otomatuis ditambahkan ke prototype


class Person {
	constructor(name) {
		this.name = name;
	}

	sayHello(name) {
		console.info(`Hi ${name}, my name is ${this.name}`);
	}
}

Catatang

Jangan menambahkan method dialam block constructor, karena hasilnay tidak di rekomendasikan itu akan menempel ke instacne object. Cara yang benara dlaah di eltakkan di didalam block class
