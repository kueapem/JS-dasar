Aggregating Modules

Ada waktunya kadang kita ingin meng agregate beberapa module menjadi satu agar penggunaannya lebih sederhana

Jika module sudah banyak, maka ketika butuh banyak kode dari banyak module, kita harus sebutkan nama module tersebut tiap kali import

Dengan menggunakan Aggregating, kita bisa membuat module agregate, yang isinya menggabungkan export dari module lain

Caranya


export {a, b, c} from "./script/module-object.js";
