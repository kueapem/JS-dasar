Membuat module

Membuat module itu sederhana, kita cukup membuat file js biasanya

Yang membedakan adalah, saat kita akan meload file JS dari HTML, kita tidak bisa menggunkaan type javascript lagi, melainkan menggunakan type module

File .js dan .mjs

Karena file js module sebenarnya sama dengan js biasa, kadang developer membedakan nama file untuk js module menggunakan .mjs

Hal ini sebenarnya tidak ada kewajibananya, hanya saja agar mempermudah ketika di tahap development, kita bisa dengan mudah memebrdakan mana file js biasa dengan file js module hanya dengan extensionnya saja

Kenapa error

Secara default, saat kita menggunakan fitur js module, semua kode baik itu fungsi, loop, variable dll tidak akan di ekspos keluar module tersebut

JIka kita ingin mengekspos kode tersebut, kita harus memberitahunya secara eksplisit
