Latar belakang

Dahulu kode js bsiasanya dimulai dari kode yang kecil dan sedikit di setiap halaman, jadi jarang sekali kita ak menggunakan kode js dengan ukuran besar dalam satu halaman website

Namun saat ini, kebutuhan web sudah berbeda, semenjak makin populernya framework js untuk membuat frontEnd, kebutuhan pembuatan kode js semakin kompleks dan besar

Oleh karena itu, sudah seharusnya saat ini, saat membuat kode js, kita perlu melakukan mekanisme memotong-motong kode js, sehingga tidak terlalu kompleks dan banyak dalam satu file JS.

Javascript modules

JS modules merupakan fitur dimana kita bisa membuat module (kode js yang bisa digunakan ulang) untuk kode js lain

Js module diawali pada teknologi bernama NodeJS, selanjutnya banyak library yang mengadopsi penggunaan js module di NodeJS agar bisa berjalan di web browser, seperti library CommonJS, AMD, RequireJS dll

Namun untungnya, saat ini browser modern sudah bnayak mendukung JS modules, kecuali jika masih menggunakan internet explorer

Import dan Export

Ada 2 kata kunci yang biasanya digunkana js module Import dan Export

Import merupakan kata kunci yang digunakan unruk mengambil sesuatu dari module lain

Export digunakan untuk memberitahu module lain, bagian mana yang ingin kita export keluar, sehingga bisa di Import oleh module lain
