Ketika sebuah web di load oleh browser, maka browser akan membuat document object model untuk halaman web tersebut

DOM dibuat dalam bentuk tree / pohon yang tiap cabang nya berisikan obejct dari jenis element HTML

JS memiliki kemampuan untuk memanipulasi DOM, artinya dengan JS, kita bisa akses tiap obeject di DOM bahkan memanipulasi objectnya
